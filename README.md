## 🚀 Proje Hakkında
**Ecemiko**, benzersiz Spider-Gwen neo-dark tasarımıyla oluşturulmuş bir Launcher / Mağaza arayüzüdür. Sıradanlığın ötesine geçerek performans ve kaliteyi en saf halinde sunmayı hedefler. Sadece oyun indirip girmek yerine, estetik zevki önceliklendirerek kullanıcıyı bir *premium deneyim* döngüsüne alır.

## ✨ Özellikler
- **🕷️ Premium Spider-Verse Tasarım Dili:** Neon detaylar, glassmorphism cam kutular, özel akıcı manyetik imleç hover efektleri.
- **📜 Akıcı "Storytelling" Animasyonu (Scroll-driven):** Siz paneli aşağı kaydırdıkça çalan devasa interaktif ekran hikayesi! Frame'ler teker teker okunarak kusursuz bir geçiş yaratır.
- **⚡ Akıllı Optimizasyon (Adaptive Scaling):** Cihazın boyutuna (mobil/widescreen) mükemmel tepki veren orantılı ekran sabitleme motoru (position sticky & intersect observer mimarisi).
- **💡 Parallax Hover Kartlar:** Fareyi hareket ettirdiğinizde 3D tepki veren kahraman katmanları ve neon vurgular (64 Bin+ oyun entegrasyonuna hazır!).

## 🛠️ Teknolojiler
Bu proje, framework gömleklerini atarak hıza ve kusursuzluğa odaklanan saf (vanilla) dillerle yazılmıştır:
* **HTML5** (Semantik Mimari)
* **CSS3** (Keyframes, Glassmorphism, CSS Variable Mantığı)
* **Vanilla JavaScript** (ES6+, IntersectionObserver, Scroll Progress Animasyonları)

## 📥 İndirme ve Ön-İzleme (Kurulum)
Sistemi doğrudan yerel sunucuda (Live Server) açabilir veya statik web sunucunuza yükleyebilirsiniz. Arayüz "tam tepkisel / responsive" özelliklere sahiptir ve herhangi bir ekstra paket yöneticisi kurmanızı (\`npm\`, \`yarn\` vb.) gerektirmez.

**Not:** Kurulum sonrasında asıl çalışacak program dosyası olan **\`Lucy V2.0.0.exe\`** (boyutu gereği) Git Reposuna dahil edilmemiştir, doğrudan **[Releases]** bölümüne yüklenebilir formattadır.
